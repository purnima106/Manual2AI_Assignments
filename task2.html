Task 2 — Convert a Messy Excel File to a Clean One (AI Data Cleaning)

Demonstrate how AI can clean inconsistent data — a repetitive but essential task in everyday work.

Estimated time
15–20 min
Difficulty
Medium
1. Manual Task
  1. Download the messy Excel sample (button on right).
  2. Open in Excel or Google Sheets.
  3. Clean the data manually by fixing:
    • Date format inconsistencies
    • Extra spaces and capitalization
    • Missing city names
    • Duplicate rows
  4. Record how long it takes to get a clean version.
2. AI-assisted Task

Upload the same sheet to any free AI tool below and ask it to clean and reformat automatically.

ChatGPT Free (file upload) Claude.ai (CSV upload)
Suggested AI Prompt
"Clean this data:
- Fix inconsistent date formats (DD/MM/YYYY)
- Remove duplicates
- Proper-case all city and name columns
- Fill missing cities with 'Unknown'
- Return the cleaned table as CSV"
