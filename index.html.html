Task 1 — Excel → Dashboard (Manual → AI)

Show participants how raw spreadsheet data becomes a presentable dashboard in minutes using free AI tools.

Estimated time: 15–20 min
Difficulty: Medium
1. Manual Task (for participants)
  1. Download the sample CSV (button on the right).
  2. Open it in Excel or Google Sheets and create: a bar chart (Revenue by Region), a line chart (Monthly Revenue trend), a scatter plot (Profit Margin vs Customer Satisfaction), and a top-performer table with profit analysis.
  3. Calculate correlation between Customer Satisfaction and Revenue, and add trend analysis for Marketing Spend efficiency.
  4. Format slide-ready visuals (titles, legends, annotations) and save as images or copy into a slide.
  5. Note how long each step took.
2. AI-assisted Task (demo)

Use one of the free web tools below. Upload the same CSV and ask the AI to build a comprehensive dashboard with multiple chart types, correlation analysis, and strategic business insights.
